Parking Lot

问题：如何设计一个停车场？

可以做如下假设：

  1. 停车场有多层，每层有多行停车位

  2. 停车场可以停摩托车，小轿车和公交车

  3. 停车场有摩托车位，紧凑型车位，和大型车位

  4. 摩托车可以停在任何位置

  5. 小轿车可以停在紧凑型车位和大型车位

  6. 公交车只能停在同一行中连续的五个大型车位上，不能停在小位置上

    public int spotsNum() {};

    protected List<ParkingSpot> parkingSpots = new ArrayList<>();

        for (int i= 0; i < parkingSpots.size(); i++) {

}
